Chaque événement a sa discussion. Messages privés entre participants pour les membres Premium.

FR · AR · EN

Interface complète en français, arabe (RTL) et anglais — mode sombre compris.

Installation en 3 étapes

Téléchargez l'application avec le bouton ci-dessus, depuis votre téléphone Android.
Ouvrez le fichier depuis vos notifications. Si Android le demande, autorisez « Installer des applications inconnues » pour votre navigateur (Paramètres → Sécurité).
Lancez Lokl, créez un compte (email ou Google) et rejoignez votre premier événement 🎉
⚠️ Version bêta de test. Cette application est en pré-production : les données peuvent être réinitialisées sans préavis. La connexion Google est limitée aux testeurs enregistrés pendant la phase de test — utilisez l'inscription par email si le bouton Google refuse votre compte.
